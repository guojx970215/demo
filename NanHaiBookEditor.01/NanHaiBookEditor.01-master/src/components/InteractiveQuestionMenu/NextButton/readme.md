### Next Button

用于在页面添加next按钮 , 点击后会跳到指定页码 , 弹框 , 点击返回指定页码

| props              | 说明                                 |
| ------------------ | ------------------------------------ |
| trans              | i18n                                 |
| visible            | 弹框的显隐控制                       |
| onCancel           | 关闭弹框                             |
| actNewAddStateNext | redex的方法, 向书籍中添加next button |
